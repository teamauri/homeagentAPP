# 5-Day Demo Plan

## Day 1 — Design system and shell

- Next.js + Tailwind setup.
- Mobile app shell with bottom navigation.
- Global CSS tokens.
- Doodle icon system.
- Implement Today skeleton.

## Day 2 — Main screens

- Finish Today, Calendar, Journeys, Moments, Family.
- Drive all screens from mock data.
- Add status pills and filters.

## Day 3 — Details and sheets

- Piano plan detail.
- Moment detail / Grandma draft.
- Add Help sheet.
- Connection detail.
- Basic transitions.

## Day 4 — Semi-real sources

- Add local media/video support for Auri Robot clip mock.
- Add Reading demo event ingestion shape.
- Keep Calendar/Photos mocked.
- Add connected/not-connected states.

## Day 5 — Polish and investor demo

- Tighten spacing, typography, responsive behavior.
- Add loading and empty states.
- Record demo flow.
- Prepare 90-second pitch script.

## Non-goals

- Real iCloud Photos sync.
- Real Apple Calendar sync.
- Real Doubao integration.
- Real auth.
- Full native app.
