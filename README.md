# Auri Family OS — Mobile Web Demo

A polished Next.js + Tailwind mobile web demo for the app-first Auri Family OS concept.

## Run locally

```bash
brew install node # only if npm is missing
npm install
npm run dev
```

Open `http://localhost:3000` and use an iPhone-sized viewport.

## What is included

- Five mock-data-driven tabs: Today, Calendar, Journeys, Moments, Family.
- Reusable app primitives: `AppShell`, bottom nav, cards, section headers, pills, row chevrons, and doodle icons.
- Investor-demo docs in `docs/`, including PRD, design system, data model, 5-day plan, and future Photos integration plan.
- Mobile-first styling with an iPhone-like desktop wrapper.

## Product direction

Auri starts as an app-first family AI operating layer. iPhone/Android native apps come later via Expo / React Native, sharing backend, types, agent logic, and design tokens with this web foundation.
